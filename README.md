# Radio

It's a radio!

What is it for? Well, I recently found a really old radio that I loved the look
of. The antenna is awful and I don't listen to normal radio, so I though I'd
stick a Raspberry Pi inside with a DigiAMP+ hat, hook it up to the speakers
themselves and then use the tuning knob on the radio to change the music that
is playing.

I wanted it to be somehow 'authentic' feel, so that's how "Radio" was born. It
let's you provide a list of long audio files that act as stations that you can
tune into, then when you are tuning it will play white noise between the stations
simulating the vibe of a real readio!
